What is grid based design?

Most webpages, if not all follow a grid structure. CSS makes it easy to setup a grid. A lot of times setting up css classes to cover 12 rows can make it entirely sufficient to design a complete website, simply by altering what row the element is placed at and controlling the width via the number of columns it spans.

Whats all the hype about responsive webpages?

Given the large number of devices being used to access the internet, it is imperative that a website be able to auto adjust for the best user experience. Using design features including @mediaquery make it easier to make a webpage 'responsive' to the screen size it is being accessed on.

What is semantic structure?

Semantic structure tells the browser about the type of elements but also makes common sense to any user viewing the code. Example 'h1' is the most prominent heading on a page but also clear that it is so to anyone reading the html/ css code.
